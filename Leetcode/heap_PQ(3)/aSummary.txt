Heap

a complete binary tree where:
(max-heap) every parent node is greater than or equal to its children.

features
- O(log n) - addElement and removeMax
- O(1) - peekMax

can be implemented by hand - an array (2*i +1 and +2 are kids)
or just use heapq in python
python heaps are min heaps by default (* -1) to use min as max heap

common uses
- need fast min/max
- priority queues
- heap sort
- dijkstras

min heap to find kth largest
max heap to find kth smallest
